Voodoo Escape 

October 2021 - November 2021
Northeastern Jam-O-Lantern 2021 submission
Theme: Curse

CLICK HERE TO SEE REPOSITORY

Physic logic

This was able to implement many of the basic concepts when programming a platformer, implementing a bit more complicated physics then your regular platformer. The implementation of wall slides, wall jumps, jumps, etc. These concepts I was able to figure out quickly. Nonetheless still ran into some challenges, but with research, patience, and testing I was able to figure them out in no time at all.

Testing Gameplay

One thing I maintain to keep track throughout this game jam was scope. There were many features that one could take with the voodoo route. With a team of 6 many ideas wanted to have their part. Keeping people on track and working and testing gameplay features that I know I could finish in the timeframe, made the game come out more polished and mad ethe gameplay for fun for the players.

Project Description

This project was a part of a two-week Game-Jam the theme of this jam was cursed. So our group landed on the idea of curse dolls. So we took that concept and moved forward with a platformer. The idea of using a cursed doll to travel throughout the terrain proposes a creative and programmable challenge that I was excited to explore.

Tools Used

  • Engine: Gamemaker studio 2
  • Programming Language: GML (based in C)
  • Source Control: Github

What I Learned

Throughout this project I gained a better understanding of gameplay mechanics. The movement code and to integrating audio and animation. Also the inclusion of fast prototyping and clear communication between team members was vital to the success of this project.
