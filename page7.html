Project Description

This program runs every permutation that is possible of a demigod team, against a specific demigod team that the user inputs. Then it spits out the win rate percent, lose rate percent, and draw rate percent of that team. This allows the developers and designers to look at these statistics and formulate methods on how to balance different units so there are not any combinations that lower the experience for the player.
